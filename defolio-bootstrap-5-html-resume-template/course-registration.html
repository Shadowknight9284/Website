Optimized Course Registration Portal

August 2024 - Present

Project Overview

A full-stack application designed to enhance the Rutgers course registration experience by providing intelligent course recommendations and automated registration features.

Key Features

  • Relational database integrating Rutgers Schedule of Classes data
  • Proprietary course difficulty rating algorithm using SIRS data
  • Automated Course Sniper with email notifications
  • Degree progress tracking and requirements visualization

Technical Implementation

  • RESTful API built with Flask for course data access
  • Real-time course availability monitoring system
  • Automated email notification service
  • Data scraping and processing pipeline for course information

Technologies Used

Python Flask SQL JavaScript HTML/CSS Bootstrap
